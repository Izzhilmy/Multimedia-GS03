# Unit 01: Project Setup

## Goal

Create a working Laravel project with dual MySQL database connections
(gs03 + mmdb2026), Blade views, and the complete folder structure.
A request to `/login` must render a Blade view before any other
unit is started.

---

## Design

No feature UI yet. This unit is entirely infrastructure.
Success is a working local dev environment with both database
connections verified and the folder structure in place.

---

## Implementation

### Laravel Installation

```bash
composer create-project laravel/laravel gender-detection
cd gender-detection
```

### Configure Dual Database Connections

Edit `config/database.php` — add `mmdb` alongside the default `mysql`:

```php
'connections' => [

    'mysql' => [
        'driver'    => 'mysql',
        'host'      => env('DB_HOST', '127.0.0.1'),
        'port'      => env('DB_PORT', '3306'),
        'database'  => env('DB_DATABASE', 'gs03'),
        'username'  => env('DB_USERNAME', 'root'),
        'password'  => env('DB_PASSWORD', '1234'),
        'charset'   => 'utf8mb4',
        'collation' => 'utf8mb4_unicode_ci',
        'prefix'    => '',
        'strict'    => true,
        'engine'    => null,
    ],

    'mmdb' => [
        'driver'    => 'mysql',
        'host'      => env('DB_MMDB_HOST', '127.0.0.1'),
        'port'      => env('DB_MMDB_PORT', '3306'),
        'database'  => env('DB_MMDB_DATABASE', 'mmdb2026'),
        'username'  => env('DB_MMDB_USERNAME', 'root'),
        'password'  => env('DB_MMDB_PASSWORD', '1234'),
        'charset'   => 'utf8mb4',
        'collation' => 'utf8mb4_unicode_ci',
        'prefix'    => '',
        'strict'    => true,
        'engine'    => null,
    ],

],
```

### Update .env

```
APP_NAME="Gender Detection System"
APP_ENV=local
APP_KEY=           # generated by php artisan key:generate
APP_DEBUG=true
APP_URL=http://localhost

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=gs03
DB_USERNAME=root
DB_PASSWORD=1234

DB_MMDB_HOST=127.0.0.1
DB_MMDB_PORT=3306
DB_MMDB_DATABASE=mmdb2026
DB_MMDB_USERNAME=root
DB_MMDB_PASSWORD=1234
```

### Generate App Key

```bash
php artisan key:generate
```

### Create gs03 Database

In phpMyAdmin or MySQL CLI:
```sql
CREATE DATABASE gs03 CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

`mmdb2026` already exists — do NOT recreate it.

### Disable Default Auth Scaffolding

This project does NOT use Laravel's built-in `users` table or Auth facade.
Do not install Breeze or Jetstream.
Remove or leave unused the default `users` migration — do not run it.

In `config/auth.php`, the defaults can remain but will not be used.
Authentication is fully manual via session (see Unit 04).

### Folder Structure

Create all required directories:

```bash
mkdir -p app/Http/Controllers
mkdir -p app/Http/Middleware
mkdir -p app/Services
mkdir -p app/Models
mkdir -p resources/views/auth
mkdir -p resources/views/detection
mkdir -p resources/views/history
mkdir -p resources/views/layouts
mkdir -p storage/app/public/uploads
```

Link storage:
```bash
php artisan storage:link
```

### Placeholder Login View

Create `resources/views/auth/login.blade.php`:

```blade
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Login — Gender Detection System</title>
</head>
<body>
    <h1>Login</h1>
    <p>Unit 04 will complete this page.</p>
</body>
</html>
```

### Placeholder Route

In `routes/web.php`:

```php
Route::get('/login', fn() => view('auth.login'))->name('login');
```

### Verify Database Connections

Create a temporary route to confirm both connections work:

```php
Route::get('/db-check', function () {
    $gs03  = DB::connection('mysql')->select('SELECT 1')[0];
    $mmdb  = DB::connection('mmdb')->select('SELECT 1')[0];
    return response()->json(['gs03' => $gs03, 'mmdb' => $mmdb]);
});
```

Remove this route after verification.

### Start Dev Server

```bash
php artisan serve
```

Navigate to `http://localhost:8000/login` — placeholder page must render.
